# Comprar

Aplicativo simples de lista de compras com filtro por status e persistência local.

## Funcionalidades
- Adicionar itens com status **PENDING**.
- Filtrar itens por status.
- Persistir itens no storage local.

## Estrutura principal
- Tela principal: [`Home`](src/app/Home/index.tsx)
- Persistência: [`itemsStorage`](src/storage/itemsStorage.ts)

## Como rodar
1. Instale as dependências:
   - `npm install`
2. Inicie o app:
   - `npm start`

> Ajuste os comandos conforme o seu fluxo (Expo/React Native).
